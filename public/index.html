Sweeten Your React Templates

Writing traditional markup?

Ok, Boomer

Collie is an indentation-based template language that compiles directly to JSX and TSX. Keep React exactly as it is — just write components faster with cleaner, more expressive templates.

  • No new runtime — just plain JSX output.
  • Works seamlessly with TypeScript.
  • First-class VS Code support.

Why teams pick Collie

Indentation-based syntax

Markup with natural indentation, no closing tags, and CSS-like selectors.

Type-safe by design

#props declarations compile to TS interfaces, inheriting React’s inference.

Zero-runtime abstraction

Collie prints plain JSX. No runtime packages. No magic.

VS Code companion

Syntax highlighting, formatting, diagnostics, and conversion tools.

How Collie fits your React stack

  1. Install the Collie CLI
  2. Author .collie files next to your TSX components
  3. The compiler emits predictable TSX that TypeScript fully understands

Tooling that ships with Collie

collie

Compiler, CLI, Vite plugin, Next adapter, AST utilities.

collie-vscode

Formatter, diagnostics, JSX ⇄ Collie conversion.

collie/cli

Scaffold, build, watch, and format your Collie templates.

Quickstart

Install dependencies 
pnpm add -D @collie-lang/cli @collie-lang/vite
Update config 
plugins: [collie(), react()]
